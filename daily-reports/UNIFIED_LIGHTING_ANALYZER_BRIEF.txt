================================================================================
                                OVERVIEW
================================================================================

AI-powered system for analyzing lighting documents, standards compliance, and 
Dialux reports. Combines multiple extraction methods with intelligent processing 
for accurate lighting system evaluation.

================================================================================
                                KEY LIBRARIES
================================================================================

PDF Processing: pdfplumber, PyMuPDF, pdfminer.six
Table Extraction: camelot-py, tabula-py, pandas
OCR: pytesseract, Pillow, opencv-python
Data Processing: numpy, scipy, scikit-learn
Web Interface: streamlit, plotly, fastapi
AI Integration: openai, transformers, torch
Database: chromadb, faiss-cpu
Development: pytest, black, flake8, mypy

================================================================================
                                LIGHTING STANDARDS
================================================================================

WHAT ARE STANDARDS?
Guidelines defining minimum lighting quality, energy efficiency, and visual 
comfort for different spaces and activities.

SUPPORTED STANDARDS:
- EN 12464-1:2021 (European workplace lighting standard)
- BREEAM (UK environmental assessment method)

HOW TO USE:
- Automatic room type mapping to standards
- Parameter-by-parameter compliance checking
- JSON-based standards database (easily extensible)

================================================================================
                                FUNCTIONALITY
================================================================================

PDF PROCESSING:
- Multi-method extraction with fallback strategies
- Advanced table extraction with quality analysis
- OCR capabilities for image-based content

DIALUX ANALYSIS:
- Three methods: Fast (2-5s), Enhanced (10-30s), Standard (30-60s)
- Room-by-room analysis with detailed metrics
- Company information extraction

STANDARDS COMPLIANCE:
- EN 12464-1 and BREEAM compliance checking
- Parameter comparison (illuminance, uniformity, UGR, power density)
- Compliance rate calculation with deviation analysis

REPORTING:
- Comprehensive recommendations with priority levels
- Critical issues identification
- Multiple export formats (JSON, CSV, Excel, PDF)
- Interactive web interface

================================================================================

================================================================================
                                INSTALLATION
================================================================================

REQUIREMENTS:
- Python 3.8+
- Java 8+ (for Camelot)
- Tesseract OCR
- 4 GB RAM minimum

INSTALLATION:
1. git clone [repository]
2. pip install -r requirements.txt
3. Install Java and Tesseract
4. streamlit run web_app.py

================================================================================
                                USAGE
================================================================================

WEB INTERFACE:
- Start: streamlit run web_app.py
- Open: http://localhost:8501
- Upload PDF and select analysis method

COMMAND LINE:
- Fast: python main.py analyze-dialux-fast --input report.pdf
- Enhanced: python main.py analyze-dialux-enhanced --input report.pdf --api-key sk-...
- Standard: python main.py analyze-dialux --input report.pdf
